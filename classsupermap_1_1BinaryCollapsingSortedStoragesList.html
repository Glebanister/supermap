Detailed Description

template<typename T, typename IndexT, IndexT RankOneSize, typename EachStorageRegister>
class supermap::BinaryCollapsingSortedStoragesList< T, IndexT, RankOneSize, EachStorageRegister >

List of SortedSingleFileIndexedStorage. For each storage, the rank is determined as log_2(SIZE / RankOneSize ). Two storages are merged if their ranks are the same.

Template Parameters
TType of storage content.
IndexTType of storage content index.
RankOneSizeSize of block which have rank 1.
EachStorageRegisterThis list storages register.
