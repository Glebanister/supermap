Detailed Description

template<typename T, typename IndexT, typename InnerRegisterInfo, typename FindPatternType>
class supermap::BinaryCollapsingSortedStoragesList< T, IndexT, InnerRegisterInfo, FindPatternType >

List of SortedSingleFileIndexedStorage. For each storage, the rank is determined as log_2(SIZE / RankOneSize ). Two storages are merged if their ranks are the same.

Template Parameters
TType of storage content.
IndexTType of storage content index.
RankOneSizeSize of block which have rank 1.
InnerRegisterInfoThis list storages registers info.
