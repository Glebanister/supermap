Detailed Description

template<typename Key, typename Value, typename IndexT, IndexT MaxRamLoad>
class supermap::Supermap< Key, Value, IndexT, MaxRamLoad >

Key-value storage. Stores all values on disk. The index is partially stored in RAM. When the index in RAM overflows, it is reset to an index on disk, where it is stored as a binary collapsible list.

Template Parameters
KeyType of key.
ValueType of value.
IndexTType of Bounds.
MaxRamLoadThe largest size of an index that can reside in RAM.
