Detailed Description

template<typename Content, typename IndexT, typename FindPattern, typename Register = FilteringRegister<Content, FindPattern>, typename RegisterInfo = typename Register::ItemsInfo>
class supermap::FilteredStorage< Content, IndexT, FindPattern, Register, RegisterInfo >

Sorted single file indexed storage with filter as a additional register.

Template Parameters
IndexTStorage order type.
FilterTFilter type. Must be an inheritor of supermap::Filter.
